# ExtendedTranslator

Story-Bakery가 배포하는 Luau 라이브러리입니다.

- 패키지 이름: `storybakery/ExtendedTranslator`
- 엔트리: `ExtendedTranslator.luau`
- 설치: `pesde add storybakery/ExtendedTranslator`

필요한 사용 예제와 추가 설명을 채워주세요.

## Tests

CSV 기반 `LocalizationTable` 테스트는 `run-in-roblox` + `tiniest` 조합으로 실행합니다.

```bash
rokit install
pesde install
lune run scripts/run_tests.luau
```

테스트 실행에는 Roblox Studio가 설치된 환경(일반적으로 Windows)이 필요합니다.
